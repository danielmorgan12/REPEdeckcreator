# REPEdeckcreator

A lightweight, browser-based deck design helper for real estate investor presentations.

## Usage

1. Run `npm start` and open http://localhost:3000 (or open `index.html` directly).
2. Enter your deal inputs, pro forma highlights, and market data.
3. Click **Generate Slide Designs** to get a slide-by-slide layout plan.
4. Use **Copy JSON** to paste the structure into your slide creation workflow.
